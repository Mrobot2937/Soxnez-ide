# Soxnez

Soxnez é um editor de código para Android com interface inspirada no VS Code, desenvolvido em Kotlin + Jetpack Compose.

## Recursos

- Interface inspirada no VS Code
- Explorador de arquivos
- Acesso a qualquer pasta do dispositivo (SAF)
- Múltiplas abas
- Destaque de sintaxe
- Numeração de linhas
- Execução de código Python
- Console integrado
- Barra de status
- Salvamento de arquivos

## Tecnologias

- Kotlin
- Jetpack Compose
- Material 3
- Chaquopy
- Storage Access Framework (SAF)

## Como executar

1. Abra o projeto no Android Studio.
2. Aguarde a sincronização do Gradle.
3. Execute em um dispositivo ou emulador Android.
